## :dart: Sobre

Projeto "Minha Rua" v2! Originalmente focado no consumo da Brasil API para localização de CEP, o projeto foi totalmente reestruturado e reestilizado em uma **plataforma Multi-Page Mobile-First** moderna e vibrante.

<br>

## :sparkles: Funcionalidades

O projeto agora atua como um canivete suíço de consultas públicas, utilizando diversos endpoints da Brasil API:

1. **📍 Busca de CEP:** Consulta detalhada de endereços (Logradouro, Bairro, Cidade, UF) com formatação em tempo real.
2. **🏢 Busca de CNPJ:** Traz informações completas sobre empresas, incluindo Razão Social, CNAE, Situação Cadastral e Formatação inteligente de endereços e telefones.
3. **🌐 Busca de Domínio (.br):** Consulta em tempo real a situação de um domínio no Registro.br, mostrando servidores DNS, datas de expiração e sugestões de domínios livres.
4. **💵 Dashboard de Câmbio:** Um painel financeiro elegante para visualizar as variações (compra e venda) das cotações de moedas internacionais em datas específicas, separado por boletins ao longo do dia.

<br>

Antes de começar :checkered_flag:, você precisa ter o [Git](https://git-scm.com) e o [Node](https://nodejs.org/en/) instalados em sua maquina (Opcional: usar a extensão [LiveServer](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) no VSCode).

## :checkered_flag: Começando

```bash
# Clone este repositório
$ git clone https://github.com/thiilins/minha-rua

# Acesse a pasta do projeto
$ cd minha-rua

# Abra no VSCode
$ code .

# Rode via Live Server ou inicie localmente o surge para deploy:
$ pnpx surge . tl-minha-rua-v2.surge.sh
```
